Chapter 4-5 JavaScript

  1. Create a json - search.html file that links to json - search.js. In your JS file , create a json variable that contains data for at least five people. Data fields should include, at a minimum: name, sex, born, died, father, and mother.
  2. Include entries for Jane Doe and Jane Ramirez.
  3. Your html page should contain a header, a text input and a Search button. When the button is clicked, search the JSON for a match. If a match is found, display the person’s information. If no match was found , display a message.
  4. Your search should be case - insensitive, so “jane doe” and “Jane Doe” are equivalent searches. You should also account for partial searches, so “Jane” should display both “Jane Doe” and “Jane Ramirez.”
  5. Challenge: Modify your JSON so that name, father and mother consist of “first” and “last” fields.
